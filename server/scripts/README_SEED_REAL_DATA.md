# دليل إضافة البيانات الحقيقية

## نظرة عامة

هذا الـ script يضيف بيانات حقيقية للعرض في التطبيق، بما في ذلك:
- **كتب** مع صور حقيقية من الإنترنت
- **إشعارات** متنوعة للطلاب
- **أخبار وفعاليات** مع صور
- **أقساط** للطلاب الموجودين

## المتطلبات

قبل تشغيل هذا الـ script، يجب أن تكون قد قمت بـ:
1. تشغيل `npm run seed` لإضافة البيانات الأساسية (طلاب، أقسام، إلخ)
2. التأكد من أن MongoDB يعمل

## الاستخدام

```bash
cd server
npm run seed:real
```

## ما يتم إضافته

### 1. الكتب (15 كتاب)
- كتب حقيقية في مجالات مختلفة:
  - علوم الحاسوب (10 كتب)
  - الرياضيات (2 كتاب)
  - الفيزياء (2 كتاب)
- كل كتاب يحتوي على:
  - عنوان ومؤلف حقيقي
  - ISBN
  - صورة غلاف من الإنترنت
  - وصف
  - عدد النسخ المتاحة

### 2. الإشعارات (6 إشعارات لكل طالب)
- إشعارات متنوعة:
  - إشعار ترحيبي
  - إشعارات الدفعات
  - إشعارات المحاضرات
  - إشعارات الاستعارات
  - إشعارات الفعاليات
- حالات مختلفة (مقروء/غير مقروء)

### 3. الأخبار والفعاليات (8 عناصر)
- أخبار (4)
- فعاليات (4)
- كل عنصر يحتوي على:
  - عنوان ومحتوى
  - صورة من Unsplash
  - تاريخ النشر/الفعالية
  - موقع (للفعاليات)

### 4. الأقساط (4 دفعات لكل طالب)
- توزيع الحالات:
  - الدفعة الأولى: مدفوعة
  - الدفعة الثانية: جزئية
  - الدفعة الثالثة والرابعة: معلقة أو متأخرة

## ملاحظات

- هذا الـ script **لا يحذف** البيانات الموجودة، بل يضيف إليها
- الصور مستخدمة من مصادر عامة (Goodreads, Unsplash)
- يمكن تشغيل الـ script عدة مرات (سيضيف بيانات مكررة)

## استكشاف الأخطاء

### خطأ: "لا توجد طلاب في قاعدة البيانات"
**الحل**: قم بتشغيل `npm run seed` أولاً لإضافة البيانات الأساسية

### خطأ: "MongoDB connection failed"
**الحل**: تأكد من أن MongoDB يعمل وأن `MONGODB_URI` في ملف `.env` صحيح

## البيانات المضافة

- ✅ 15 كتاب جديد مع صور
- ✅ 6 إشعارات لكل طالب
- ✅ 8 أخبار/فعاليات
- ✅ 4 دفعات لكل طالب
